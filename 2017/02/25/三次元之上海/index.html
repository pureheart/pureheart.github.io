三次元，来魔都了！

发表于 2017-02-25   |   分类于 产品   |     阅读次数
用心去做，每一个经历都会变成一颗珠子，最终连成一串手链，璀璨夺目

前言

上周开始，与三节课的缘分，一发不可收拾。从申报3.3计划到线上产品进阶课程，再到线下三次元活动，带给我不止是惊喜与期待，还有各种思维碰撞和奇妙经历的体验感受，我仿佛又看到了那一束光，照亮的不只是我一个人的心房。

背景

作为产品狗、运营喵，相信你有极大的可能在某一瞬间邂逅过三节课的文章。而我，亦然如此。与三节课熟识的缘分，要从Luke写的《产品的视角》上的一个错别字说起（请见下一篇：我与三节课的缘分，从一个错别字开始）。本篇文章，分享今日参加的由简运营承办的三次元上海线下活动的经历与感受。

三次元

三次元是三节课旗下的线下分享沙龙，采用启发性讨论学习的模式，每期都由三节课志愿者利用业余时间，热情策划举办。此次是三次元首次降临上海！作为参与者，激动的同时，面对魔都尴尬的互联网氛围还是忧桑不已，因为在上海此类活动寥寥无几。

报名

成功申请加入第四期3.3计划之后，开始更深入的了解三节课。此时三次元恰好落户上海，遂马不停蹄的申请报名。有趣的是，报名采用邀请码机制，而网站上报名入口有关获取邀请码的提示有误，导致差点错过，这里要感谢三节课贺妤琪同学的帮助。此次三次元具体是怎么玩的？

准备

看到Ted talk就兴奋，可是再看看“分享一下在2016年的学习或工作中你遇到的困难，并且这个困难，你是通过什么方式、方法解决的？”，内心百感交集，各种思绪和经历涌上心头，一时间反而找不到哪一件特别刻苦铭心的经历。而这或许恰恰是三节课的用意，他逼着你去发现去总结，你经历的点滴。
周五下班后回到家开始思考，从9点开始到12点，一直抓腮挠耳，最终基本准备完毕时，已经凌晨四点（恩，凌晨四点的张江我已经不止一次看到过了）。
“师傅，你开慢点，没事的”–第二天赶赴会场的出租车上，我还在改keynote…

开始

经过几次迂回，终于从迷宫一般的楼宇中找到场地，然而保安要求需要有会场朋友带领才许上楼！！！命途多舛，乖乖绕道，从侧门3W咖啡进入。推开3W咖啡大门询问是否可以从这里上5楼时，被亲切的告知“521吧？这边直走，电梯直达”，互联网思维指导下的服务意识可见一斑！
“腼腆的我”，只能签到完，边等待边修改keynote。不出所料，活动推迟15分钟才开始（终于知道报名入口，三声五令不要迟到的意义了）。

分享

主持人邢子简单介绍后便正式开始，安仔（简运营负责人，第四期3.3计划学员）带领大家游戏热身，然后进入分享交流模式。
第一个是，是我…而且，Mac没法直接投影，意味着花了大量时间践行美貌大王对于神奇移动的教导，无法展示了。
以下，按照出场顺序摘录了大家分享的部分内容（有修改）。

我@小易 汉纸

沪江产品，三节课第4期准学员。
初入职场，一瞬间想要回学校读研，一瞬间想要转行做技术。
自述了从创造性、自由度、系统化三个方面，对以上两个问题进行分析，并通过一些方式解答。

收获：

  • 继续保持好奇心，stay hungry stay foolish。
  • 沉下心来，做好当下，迎接美好未来！

@盼盼 妹纸

基金运营，对外汉语专业。
在罗马尼亚任汉语志愿者期间，一个人去周边好几个国家游历了十多天。

  • 学会改变环境，改变自身的磁场
  • 主动选择
  • 少而精
  • 宇宙每天都在变
  • 有爱在，一切便都在，深爱身边每一位朋友
  • 坚持涂防晒霜（从此，此梗深耕上海三次元）

@管大宇 汉纸

Ued讲师、创业公司产品。
15岁开始，一个人出国留学。
大宇同学给大家扎了一个假帐篷。

  • 加入创业团队，第一周走了四个小伙伴
  • 产品经理谈运营方式，是不是属于狗拿耗子的行为呢？（不是，这里推荐阅读一个优秀的运营，到底需要有多懂“产品”？

@恒子 妹纸

沪江西班牙语运营&内训讲师，营销大师高建华关门弟子。
曾脱稿主持嗨翻700人的客户晚宴。

  • 出生教师之家，极度热爱教育，携带简历参加沪友聚会搞定沪江
  • TFS超级密室&迪士尼公主体验 带来的极致体验的思考
  • 性感工具—完整产品模型：核心产品（功能与性能）→ 外围产品（服务与配套设施）→ 外延产品（体验与感觉）

@赵超群 汉纸

用户运营，擅长典型用户分析，成功打造20W用户付费购买产品。

  • 产品和运营边界会越发模糊：未来只会有三个层面，决策层（战略层）、增长层、执行层
  • 低价策略（一元门票），疯狂增长，但复购率极低
  • 通过9.9元年卡提高购买门槛，解决复购率问题（购卡用户复购率是普通用户的三倍）
  • 年卡冷启动之后，人数上不去（用户被push拉起购买下单时，发觉价格全年不变，取消等待下次需要时再购买）
  • 推出不同时段差异化价格（三节课课程限时开放也是一个道理，督促用户当下学习，而非先报名下次再学）
  • 商业运转导致产品需要下架，因用户投诉，最终赔偿退款（用户体验的重要性）

@张佳风 汉纸

择业ing，做过产品&内容运营、app推广。
干过完成了13万字的科幻小说，等待发表。
分享科幻创作中的困难与烦恼。

  • 科学与理论的缺乏：针对性训练（推荐《刻意训练》）、吸收消化（不可求急→产生自己的理解）
  • 碎片化创作：充分构思，边写边改
  • 不自信，自我否定：坚持自我
  • 初稿完成后的困惑：小范围试读、不一定立即改正
  • 保持好奇心，保持执行力，保持接地气
  • 推荐《你一生的故事》&电影《降临》

@星河 汉纸

网线内容运营，16年组织了两场百人线下活动并深受好评，微作业被亮哥发布。
分享了自去年六月步入互联网行业后的摸索历程。

@邹晓宁 妹纸

16年换了4份工作，5个小时跑完全程马拉松。

  • 想学设计，被家人拒绝，跑完全马和蹦极之后，发现依然热爱设计
  • 跨专业报考天津大学，老师同学不看好，但全情投入，但终因身份证丢失，无疾而终
  • 四份工作，都比较奇葩，但依旧乐观

邹晓宁，经历了各种挫事，最终选择来到上海，寻找那份希望。

@刁海峰 汉纸

渠道运营，在学校获得过省银奖，从学校到职场转化极快。

  • 能够记住每个人的名字
  • 放荡不羁，走南闯北
  • 学校首个人人网校园大使
  • 负责校园首个公众号，粉丝数超2W，最高到校园公众号前8
  • 只要用心，加了N多大咖的微信，一封邮件不行那就两封
  • 胖是有原因的：拖延症
  • 目标一年阅读：50本，已坚持3个月，共15本书

@宋小错 妹纸

洋码头活动运营，极度热爱现在的工作。

  • 从非盈利面向大学生网站，跳槽电商
  • 加入了网易云课堂学习，疯狂投简历
  • 从买买买，到专注海购电商
  • 日本旅游，设计师朋友全程地陪，自己全程介绍国内海购环境等，成功让朋友入住洋码头，并吸引众多客户
  • 会不断尝试优化购物流程，比如砍价后，直接购买而非等待时限后付款

@大焦 妹纸

数据分析，加入过风投公司，热爱推理小说。

  • 经历了职场危机，三观被毁
  • 面对对与错的纠结，选择了自己的态度，而非他人的非议
  • 经历风雨磨砺，直面弱点，谈笑风生
  • 每天全情投入浦东图书馆，深钻数据分析
  • 身怀感恩：回想曾经两家公司的Boss把自己折磨不堪，如今终领悟了教导
  • @恒子 面对职场危机，要么走，要么有一技之长保全自身

@洋洋 汉纸

新媒体渠道管理，本次活动摄影师。
搭建了公司新媒体品牌推广的整套方案，从互联网菜鸟成长为企业得力助手。

  • 切勿为了忙而忙，找准自身定位和价值
  • 年三十小侄子诞生，带了了对于自身认知的思考：从6个月有自身与母亲是分开的意识，到15岁思考独立的意义，再到25岁寻找意义
  • 人是环境的产物，挑选环境，获取更多正反馈
  • 保持对于兴趣的热爱和投入：摄影
  • 简运营组织带来了工作之外的意义和兴趣
  • 跳出舒适区：尝试卖茶
  • sign：帮助梳理成长中经历的事情，找出长处与热爱

以七年为一个单位，从四个方面（sign）梳理：

  • success：自我效能
  • instinct：自动自发
  • grow：学的很快（有的人对理念更感兴趣，有的人对于执行更感兴趣）
  • need：事后满足感爆棚（享受过程）

收获

  • 勇敢去尝试，发现热爱，全情投入，便会铸就最好的自己。
  • 历经逆境，或许会深陷迷潭，但终将会风轻云淡，经历过你便成长。
  • 永远相信，美好的事情终将发生！
